Send Bulk order confirmation receipt to the Customers.

== Description ==
WooCommerce Bulk Receipt plugin allowed an admin to send bulk order confirmation receipt to their customers.

Currently, If WooCommerce store owner wants to send again Order Confirmation to their Customer then they have to go each order and re-send Customer receipt. In Case, If there are a number of customer who did not get Confirmation receipt then it's very time consuming to re-send them.

So by WooCommerce Bulk Receipt plugin, Store Owner can send bulk Order confirmation receipt to their customers.

= Features: =
* Send Bulk Order Confirmation receipt to Customers.
* Easily customizable success message
* Localization Support

== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress 4.4+
* WooCommerce

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install of WooCommerce Bulk Receipt, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "WooCommerce Bulk Receipt" and click Search Plugins. Once you have found our plugin you can install it by simply clicking Install Now. After clicking that link you will be asked if you are sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application.

<ol>
<li>Download the plugin file to your computer and unzip it</li>
<li>Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.</li>
<li>Activate the plugin from the Plugins menu within the WordPress admin.</li>
</ol>

== Frequently Asked Questions ==

= Q. Why I need this Plugin? =
Ans: In case, if your customers did not get order confirmation receipt then you can easily send them by using this Plugin.

= Q. How do I send Bulk Order Confirmation Receipt to our customers? =
Ans: Please go to WooCommerce Order listing page, You will find 'Email invoice / order details to customer' option in Bulk Option. By this way, You can send bulk confirmation receipt to your customers.
